# Chat Socket Application Using Cipher-Api DES

Using Socket programming and threading to make a real-time chatroom in c 

## Requiment-: 
1. ubuntu 18.04
2. make
3. gcc++

### Usage script 

* build and install `des_module`

```bash
cd encrypt_module
make
make install
```

### Usage Server script 

* in root project

```bash
make server
./server
```

### Usage Client script 

* in root project

```bash
make client
sudo ./client
```
